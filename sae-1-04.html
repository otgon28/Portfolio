AC13.01 AC13.04
Contexte & objectifs

Dans ce projet, nous avons d'abord réalisé deux TPs d'entraînement. Le premier consistait à reproduire au plus près un visuel fourni par le professeur, et le second portait sur des exercices de rappel sur Flexbox, CSS Grid et le responsive design. Ensuite, pour être noté, nous devions créer un site Web multipages sur un thème libre.

Travaux réalisés
1
Entraînement au Responsive Design

Exécution des deux TPs d'entraînement pour maîtriser la reproduction d'un design et la gestion du responsive design.

2
Conception et développement du site

Structuration des pages en HTML5 sémantique, mise en forme avec CSS3. Découverte du modèle de boîte, du positionnement et de la mise en page responsive.

3
Git et Github

Utilisation de Git et GitHub pour versionner le code et héberger le site sur GitHub Pages.

Preuves & traces produites

Les traces de ce projet montrent le site web réalisé, son organisation en plusieurs pages, la mise en page responsive et les choix visuels effectués pendant le développement.

Voir le site web publié sur GitHub Pages
Résultat

Ce projet m'a permis de construire un premier site web complet et de mieux comprendre le fonctionnement d'un site, depuis la structure HTML jusqu'à la mise en forme CSS et l'hébergement sur GitHub Pages.

Auto-évaluation

Ce projet a renforcé mes bases en développement web. J'ai progressé dans l'organisation du code, la mise en page et la compréhension des échanges entre navigateur et serveur. Les difficultés principales étaient liées à la cohérence visuelle et à l'adaptation de l'affichage selon les tailles d'écran.

Problème rencontré Solution envisagée ou envisageable
Difficulté à obtenir une mise en page propre et lisible. Utiliser une structure HTML claire et tester régulièrement le rendu dans le navigateur.
Problèmes d'adaptation sur différentes tailles d'écran. Demander de l’aide au professeur et revoir les cours sur le responsive design
Manque de temps pour terminer toutes les pages du site Planifier le temps de travail et prioriser les tâches.
